# 📱 Flipkart Smartphone Competitive Intelligence: Data-Driven Market Analysis

## 🚀 End-to-End Data Analytics Project

> **Executive Overview:** This project delivers a high-impact **Competitive Intelligence Audit** of the Indian smartphone market using real-time data scraped from Flipkart. By engineering a custom "Popularity Index" and analyzing discount elasticity across **449 product listings**, this study identifies the core battlegrounds for 6 major global manufacturers and provides actionable insights into consumer demand patterns.

## 🔄 Project Workflow

### 1️⃣ Data Collection
Scraped real-time smartphone listing data directly from Flipkart.

### 2️⃣ Data Cleaning (Python)

- Removed missing values  
- Standardized brand/model names  
- Fixed data types  
- Prepared clean analysis-ready dataset  

### 3️⃣ Feature Engineering

Created custom columns to improve analysis quality and business insights:

- **Price Category** – Derived from the **MRP column** to classify products into Budget, Mid-range, Premium, and Luxury segments  
- **Popularity Score** – Created using **Ratings + Reviews** to estimate market demand and customer interest  

### 4️⃣ Data Analysis (SQL)

Generated business insights using SQL queries.

### 5️⃣ Dashboard Creation (Power BI)

Built interactive visuals for decision-making.

## 🎯 Objective (EDA)

Exploratory Data Analysis was performed to answer key business questions such as:

- Identify leading smartphone brands  
- Analyze Budget, Mid-range, and Luxury segments  
- Understand customer preferences  
- Measure the impact of discounts on popularity  

---

## 📈 Key Insights

- **Apple** leads in popularity and ratings  
- **Samsung** performs strongly across all segments  
- **Realme & Redmi** dominate the budget market  
- **Mid-range** is the largest segment (**46.88%**)  
- **8GB RAM** and **128GB/256GB Storage** are the most common configurations  

---

## ⚠️ Limitations

- Apple listings on Flipkart often do not provide **RAM** and **Battery Capacity** details.  
- Comparisons involving these features across brands may not be fully accurate.  
- Insights related to Apple in these categories are based on available listing data and customer market response.  
- Listing visibility may vary by region or popular area, which can affect the number of products shown for each brand.
